# igxjs
A collection of practical JavaScript/Node.js open-source projects for everyday development needs, maintained by the igxjs organization on GitHub.

## About igxjs
igxjs curates and develops lightweight, functional Node.js/JavaScript tools and components designed to streamline common development workflows. Our projects focus on **simplicity**, **usability**, and **compatibility**, with clear licensing and regular maintenance to boost developer productivity.

## Licensing
Our projects adopt two industry-standard open-source licenses to match different use cases, all free for commercial and non-commercial use per the respective license terms:
- **Apache License 2.0**: For organization core configuration and shared ecosystem components
- **MIT License**: For standalone utility tools – permissive licensing with minimal restrictions

## Maintenance
All igxjs repositories are actively maintained and regularly updated. We prioritize compatibility with the latest Node.js versions and address issues and feature requests in a timely manner to ensure our tools remain reliable for your projects.
